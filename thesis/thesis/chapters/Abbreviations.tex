\chapter*{List of Abbreviations}

$C_\text{properties}$ container properties

$S$ container sizes

$W_\text{max}$ maximum allowed container weight

$N$ number of input boxes

$\rho$ rotation function

$\mathcal{R}$ set of admissible rotations

$\mathrm{V}$ volume function

$B$ list of box properties

$B_i$ properties of box $i$

$s_i$ size of box $i$

$w_i$ weight of box $i$

$p_i$ position (start coordinate) of box $i$

$r_i$ region occupied by box $i$

$\mathcal{C}$ set of opened containers

$\mathcal{B}_c$ set of boxes placed in container $c$

$W_c$ total weight of container $c$

$\mathrm{V}(c)$ occupied volume of container $c$

EMR empty maximal region

$\mathcal{E}_c$ set of EMRs in container $c$

$\mathcal{H}$ set of placement heuristics

$\Pi$ packing rules

$o$ permutation (packing order) of boxes

$\boldsymbol{\rho}$ rotation assignments for boxes

$\mathbf{h}$ heuristic assignments for boxes

$M$ population size

$G$ number of generations

$p_E$ proportion of elite individuals

$p_R$ proportion of random individuals

$p_C$ crossover probability (inheritance from elite parent)

$p_M$ mutation probability

$I_{HC}$ number of hill-climbing iterations

$p_{HC}$ hill-climbing mutation probability

$p_A$ probability of accepting a worse solution

$p_A^{(0)}$ initial probability of accepting a worse solution

$\alpha$ geometric decay coefficient for $p_A$

$p_{\min}$ minimum acceptance probability
